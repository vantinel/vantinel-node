# Changelog

All notable changes to Vantinel JS/TS SDKs are documented here.

## @vantinel/node-sdk

### [0.4.0] - 2025-02-01
- VantinelMonitor with `watchTool()` wrapper for automatic instrumentation
- Async fire-and-forget telemetry
- Session management with automatic ID generation
- Cost estimation and token counting
- Circuit breaker for graceful degradation

## @vantinel/js-sdk

### [0.4.0] - 2025-02-01
- Browser-compatible Vantinel client
- CSP-safe telemetry with minimal footprint
- ESM + CJS dual build

## @vantinel/nextjs

### [0.2.0] - 2025-02-01
- Next.js App Router integration
- Server-side and client-side monitoring components
- Automatic middleware instrumentation
